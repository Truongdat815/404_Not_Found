# Frontend - Requirements Engineering Dashboard

## Tech Stack
- React + Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Axios

## Setup

```bash
cd frontend
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```
